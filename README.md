# 🎮 Tic-Tac-Toe Web Application

A fully interactive **Tic-Tac-Toe web game** built using **HTML, CSS, and JavaScript**, with support for:

* 🤖 AI opponent
* 👥 Multiplayer (Socket.io)
* 📊 Scoreboard tracking
* 🏆 Win animations
* 📜 Game history (local storage)

---

## 🚀 Live Demo

* 🌐 Frontend (Netlify): https://tictactoewebgamebyshubh.netlify.app/
* ⚙️ Backend (Render): https://prodigy-wd-03-1.onrender.com

---

## 📌 Features

### 🎯 Core Features

* Play Tic-Tac-Toe in browser
* Player vs Player mode
* Player vs AI mode
* Restart game functionality

### 🤖 AI Mode

* Random AI move generation
* Automatic opponent response

### 👥 Multiplayer Mode

* Real-time gameplay using Socket.io
* Create and join rooms

### 📊 Scoreboard

* Tracks:

  * Player X wins
  * Player O wins
  * Draws

### 🏆 Win Animation

* Winning cells highlight with animation

### 📜 Game History

* Stores results using browser localStorage
* View past matches with timestamps
* Clear history option

## 🎮 How to Play

1. Select game mode (AI / Multiplayer)
2. Click on grid to place your mark
3. First player to align 3 marks wins
4. Use Restart button to play again
